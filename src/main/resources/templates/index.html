ゲーム参加型日程

ゲームの予定、フレンドとの参加管理、募集掲示板をひとつにまとめたスケジュール管理アプリです。 ログイン前でもサービス概要、使い方、規約を確認できます。

Overview

アプリの概要

まずはアプリの概要だけを短く伝え、詳細は必要なページだけに分けています。

予定作成 日付・ゲーム・参加者を整理
フレンド管理 検索・申請・確認をまとめて管理
募集掲示板 参加募集と承認をシンプルに運用
Features

主な機能

短い説明を中心に構成しています。

SCHEDULE

予定の作成・共有

日付、ゲーム名、参加者、メモをまとめて管理できます。

FRIENDS

フレンド管理

フレンド検索、申請管理、プロフィール確認ができます。

BOARD

募集掲示板

参加募集スレッドの作成と承認ができます。

PROFILE

プロフィール設定

アイコンやリンクを設定して、自分の情報を整えられます。

How to

使い方

最初の利用手順だけを、3つに絞ってまとめています。

1

ログインする

アカウントでログインして機能を使います。

2

予定を登録する

ゲーム名や日付、参加メンバーを入力して保存します。

3

共有して集める

共有リンクや募集掲示板で参加者を集めます。
